## 🚀 Projects

### 🏏 [IPL Score Predictor](https://github.com/MANVIS10/ipl-score-predictor) — [Live Demo](https://ipl-score-predictor-mwzfkvcke36kk87jwjkqyy.streamlit.app)
Predicts IPL innings final score from current match situation using Random Forest Regressor trained on 1.7 lakh ball-by-ball records (2008–2019). Deployed as an interactive Streamlit web app.
> Python · Scikit-Learn · Streamlit · Pandas · Pickle

### 👗 [AuraStyle Predictive Pipeline](https://github.com/MANVIS10/Aurastyle-predictive-pipeline)
Climate-driven outfit recommendation system that fetches live weather data via REST API and routes it through a Decision Tree Classifier to suggest outfits. Connected to a MySQL backend for inventory management.
> Python · Scikit-Learn · Streamlit · MySQL · REST API

### 📡 [IoT Hardware Telemetry Pipeline](https://github.com/MANVIS10/IOT-pipeline-project)
Automated data pipeline that monitors real-time hardware health metrics — voltage, current, and temperature — from a MySQL database. Flags thermal failures and voltage drops using NumPy analytics and visualizes degradation patterns.
> Python · Pandas · NumPy · MySQL · Seaborn
